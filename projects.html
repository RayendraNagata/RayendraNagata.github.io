Eventive

Full Stack Developer

A comprehensive SaaS event management platform that empowers event organizers to create, manage, and scale their events with professional-grade tools. Eventive streamlines everything from small workshops to large-scale conferences with advanced ticketing, attendee management, and real-time analytics.

Key Features

  • Event Creation & Management with comprehensive scheduling and capacity management
  • Advanced Ticketing System with multiple ticket types and payment processing integration
  • Digital Ticket Generation with QR code-based validation and real-time check-in
  • Professional Check-in Interface for streamlined attendee validation
  • Analytics Dashboard with comprehensive insights on sales, attendance, and engagement
  • Public Event Discovery with searchable directory and advanced filtering

Problem & Solution

Event organizers often struggle with fragmented tools for event management, complex ticketing processes, limited attendee engagement features, and poor analytics capabilities. Eventive addresses these challenges by providing a unified management dashboard, streamlined ticketing with multiple payment options, real-time analytics and reporting, and professional check-in interfaces for seamless attendee experiences.

Technical Implementation

Built with Next.js 14 and TypeScript for the frontend, featuring modern App Router and NextAuth.js for authentication. The backend utilizes NestJS with Prisma ORM and PostgreSQL for robust data management. Integrated with Stripe and Midtrans for payment processing, and designed with enterprise-grade security and scalability in mind.

Technologies Used

Next.js 14 TypeScript NestJS Prisma PostgreSQL NextAuth.js Tailwind CSS Stripe Midtrans JWT

Quicacademy

Lead Full Stack Developer

An intelligent educational platform designed to transform learning through AI-powered content generation and personalized study experiences. Quicacademy leverages artificial intelligence to provide adaptive learning experiences for students across all educational levels.

Key Features

  • Smart File Processing with multi-format support and automatic text extraction
  • AI-Powered Learning with intelligent summarization and dynamic quiz generation
